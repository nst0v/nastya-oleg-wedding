# Wedding Website

Мобильный сайт-приглашение на свадьбу с анкетой, которая отправляет ответы в Telegram напрямую из браузера.

## Важно про безопасность

Этот вариант работает на GitHub Pages без Vercel, но токен Telegram-бота будет лежать в публичном `script.js`. Любой человек сможет открыть исходный код сайта и увидеть токен.

Для свадебной анкеты это может быть приемлемо, если вы понимаете риск. После мероприятия токен лучше перевыпустить или удалить бота.

## Структура

- `index.html` - контент, дата, место, фотографии и форма анкеты.
- `styles.css` - палитра, типографика, экранные секции и анимации.
- `script.js` - плавное появление блоков, обратный отсчет, активная навигация и отправка анкеты в Telegram.

## Что заменить

- Имена и тексты - в `index.html`.
- Дату для обратного отсчета - в `data-wedding-date` у `.hero`.
- Фото - замените блоки `.photo-placeholder` на теги `img` в нужных секциях.
- Адрес и ссылку на карту - в секции `#details`.
- Токен Telegram-бота - в `script.js`, переменная `TELEGRAM_BOT_TOKEN`.

## Telegram-анкета без Vercel

### 1. Создать или перевыпустить токен бота

Если токен уже где-то публиковался, лучше перевыпустить его.

1. Откройте Telegram и найдите `@BotFather`.
2. Для нового бота отправьте `/newbot`.
3. Для перевыпуска токена отправьте `/revoke` и выберите нужного бота.
4. Скопируйте новый токен вида `1234567890:AA...`.

### 2. Получить `chat_id`

1. Откройте созданного бота в Telegram.
2. Нажмите `Start` или отправьте ему сообщение `test`.
3. Откройте в браузере ссылку:

```text
https://api.telegram.org/botВАШ_ТОКЕН/getUpdates
```

4. Найдите в ответе:

```json
"chat": {
  "id": -1003934063475
}
```

Это значение нужно вставить в `TELEGRAM_CHAT_ID`.

### 3. Проверить токен в коде

Откройте `script.js` и найдите:

```js
const TELEGRAM_BOT_TOKEN = "...";
const TELEGRAM_CHAT_ID = "-1003934063475";
```

Токен уже прописан в `TELEGRAM_BOT_TOKEN`.

`TELEGRAM_CHAT_ID` уже выставлен на `-1003934063475`, чтобы анкеты приходили в канал. Если хотите получать анкеты в другой чат или группу, замените это значение.

## GitHub Pages

1. Загрузите файлы в репозиторий.
2. Откройте `Settings` -> `Pages`.
3. Выберите ветку `main` и папку `/root`.
4. Сохраните настройки.

После обновления GitHub Pages анкета будет отправлять данные напрямую в Telegram Bot API.
