# Wedding Website

Мобильный статичный сайт-приглашение для GitHub Pages.

## Структура

- `index.html` - контент, дата, место, фотографии и форма анкеты.
- `styles.css` - палитра, типографика, экранные секции и анимации.
- `script.js` - плавное появление блоков, обратный отсчет, активная навигация и демо-анкета.

## Что заменить

- Имена и тексты - в `index.html`.
- Дату для обратного отсчета - в `data-wedding-date` у `.hero`.
- Фото - замените блоки `.photo-placeholder` на теги `img` в нужных секциях.
- Адрес и ссылку на карту - в секции `#details`.

## GitHub Pages

1. Загрузите файлы в репозиторий.
2. Откройте `Settings` -> `Pages`.
3. Выберите ветку `main` и папку `/root`.
4. Сохраните настройки.

Анкета сейчас сохраняется в браузере гостя через `localStorage`. Для реального сбора ответов подключите Formspree, Tally, Google Forms или собственный backend.
